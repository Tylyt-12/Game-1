# HTML / JavaScript Starter Project

Open `index.html` in your browser or use a local server extension such as Live Server.

## Files

- `index.html` — main page
- `styles.css` — styling
- `script.js` — JavaScript interactions

## Quick start

1. Open this folder in VS Code.
2. Open `index.html`.
3. Use Live Server or open the file directly in a browser.
